[官网](https://www.jenkins.io/)

* 安装lts版本
    ```
    brew install jenkins-lts
    ```

* 常用操作
    ```
    启动： brew services start jenkins-lts
    重启： brew services restart jenkins-lts
    更新： brew upgrade jenkins-lts
    ```
